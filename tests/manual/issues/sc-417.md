---
title: "#417 Container after raw HTML selects both"
---

Clicking a pattern instance or container placed AFTER a raw-html block used
to select the raw block too (tree showed both, sidebar showed the HTML
card): the unprevented click let Chromium park a phantom caret at the
nearest text position — the raw block's — and any pixel of drag promoted it
into a cross-block run. Surface clicks now own the gesture end to end.

## Checks

- [ ] Click the pattern's surface (the gap between its heading and the columns) — ONLY "Two testimonials" selects: one tree highlight, pattern card in the sidebar, no caret anywhere.
- [ ] Repeat sloppily — press, wiggle the mouse a pixel or two, release — still only the pattern.
- [ ] Same two checks on the Group placed after the second raw block: only the Group selects.
- [ ] Click INTO the pattern's heading text — normal caret editing, no block selection.
- [ ] Deliberately drag from the middle of a raw block into the pattern — the whole run (raw + pattern) selects, as before.

## Fixture

```html
<blockquote>Raw markup without annotations — an opaque raw-html block.</blockquote>
<div
  data-pb-block="pattern"
  data-pb-pattern="testimonials"
  data-pb-pattern-version="1.0"
  data-pb-children
>
  <h2 data-pb-block="heading" data-pb-tag="level" data-pb-rich="text" class="text-center">
    What people say
  </h2>
  <div data-pb-block="columns" data-pb-children>
    <div data-pb-block="column" data-pb-children>
      <blockquote data-pb-block="quote">
        <div data-pb-rich="body">Exactly the tool I did not know I needed.</div>
        <cite data-pb-text="citation">Alex, maker</cite>
      </blockquote>
    </div>
    <div data-pb-block="column" data-pb-children>
      <blockquote data-pb-block="quote">
        <div data-pb-rich="body">Set up in minutes, shipped the same day.</div>
        <cite data-pb-text="citation">Sam, founder</cite>
      </blockquote>
    </div>
  </div>
</div>
<blockquote>A second raw block, directly before a plain group.</blockquote>
<div data-pb-block="group" data-pb-tag="tag" data-pb-children>
  <h3 data-pb-block="heading" data-pb-tag="level" data-pb-rich="text">Group after raw HTML</h3>
  <p data-pb-block="paragraph" data-pb-rich="body">
    Click the gap between this paragraph and the heading.
  </p>
</div>
```
