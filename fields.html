Inline fields

Three independent fields, each backed by its own PublrEditor instance — canvas only, no chrome. Type in one and watch only its stored value change; undo (⌘Z) is scoped to the field holding the caret. On an empty block, type "/" for the quick picker or click its + to insert a block.
