# Social-link icon sources

One SVG per service; the filename is the service slug. `npm run icons:social`
regenerates `src/blocks/social-icons.ts` from these files.

- `chain.svg`, `mail.svg`, `feed.svg` — original Publr artwork (house style).
- All brand marks — path data from [Simple Icons](https://simpleicons.org),
  CC0 1.0 (public domain). The logos remain trademarks of their respective
  owners, used nominatively to identify the linked service.
